# Astra

## Features

- Dual Mono To Stereo I/O
- **Stereo FX**:
  - Tape Delay
  - Ping-Pong Delay
  - Freeverb
  - C Reverb (Lukas)
  - Chorus
  - Flanger
  - Phaser
  - AirWindows
